# Rules

- **Tiêu chuẩn code**: Một file code nguồn không được vượt quá 500 dòng. Nếu file vượt quá 500 dòng, cần chia nhỏ thành các component, module hoặc helper độc lập.
- **Quản lý phiên bản Git**: Cấm tự ý thực hiện commit (`git commit`) hoặc push (`git push`) mã nguồn lên server. Chỉ thực hiện chỉnh sửa, tối ưu hóa code và chạy các lệnh test/build tại local. Việc commit/push code sẽ do USER tự thực hiện.
